## About This Project

Luxe Store is not a template. It is a **complete, handcrafted e-commerce frontend** designed from scratch with obsessive attention to detail. Every component, every animation, every color gradient was intentionally chosen to create an experience that feels premium, modern, and trustworthy.

This is the kind of interface you see at companies that spend millions on design — built here with open-source tools and a relentless focus on craft.

## Premium Design Philosophy

### Visual Identity
The entire application follows a **cohesive indigo-blue color system** built on CSS custom properties. Light and dark modes are both supported with seamless transitions. The color palette was chosen to evoke trust, sophistication, and modernity — the same psychological triggers used by Apple, Stripe, and Linear.

### Typography & Spacing
- **Inter** typeface for clean, modern readability at every size
- Consistent **8px spacing grid** across all components
- **12–16px border radius** on every card, button, and input for a soft, premium feel
- Carefully tuned **font weights** — extrabold for headlines, semibold for labels, regular for body

### Shadows & Depth
Every shadow in this project is a **multi-layered custom shadow** — not a single `box-shadow` value. Cards, buttons, and modals use 3-layer shadows that create realistic depth:

```
0 0 0 1px rgba(0,0,0,0.03)     — subtle border
0 8px 16px rgba(0,0,0,0.06)    — mid shadow
0 24px 48px rgba(0,0,0,0.08)   — ambient shadow
```

This is the same shadow technique used by Apple's UI framework.

### Animations & Interactions
- **Framer Motion** powers every page transition, card entrance, and micro-interaction
- Staggered fade-in animations on grids create a cinematic reveal effect
- Cards lift on hover with spring-physics easing (`cubic-bezier(0.34, 1.56, 0.64, 1)`)
- Buttons have a **shine sweep** effect on hover — a subtle light reflection that signals interactivity
- Mesh gradient hero sections with **floating animated orbs** create depth and movement

### Gradients & Mesh Backgrounds
Hero sections use **CSS mesh gradients** — multiple overlapping radial gradients that create a rich, dimensional background. These are not simple linear gradients; they are the same technique used in Apple's marketing pages and Vercel's dashboard.

### Glassmorphism
Navigation and overlays use `backdrop-filter: blur(20px) saturate(180%)` with semi-transparent backgrounds — creating the frosted glass effect popularized by iOS and macOS.

## Features

### Complete E-Commerce Flow
- **Product Catalog** — 100+ curated products with images, ratings, reviews, and pricing
- **Advanced Search** — Real-time search with autocomplete suggestions and search history
- **Category Browsing** — Filter by category, brand, price range, and rating
- **Product Detail** — Full product pages with image gallery, size/color selection, reviews, and related products
- **Shopping Cart** — Add, remove, update quantities with real-time totals
- **Checkout** — Multi-step checkout with address form, payment selection, and order summary
- **Order Confirmation** — Animated success page with order details

### User System
- **Authentication** — Login, register, and forgot password flows
- **Dashboard** — Order history, profile management, notifications, and wishlist
- **Persistent State** — All data survives page refresh via localStorage

### UI/UX Excellence
- **Dark Mode** — Full dark mode with one-click toggle
- **Responsive Design** — Optimized for mobile, tablet, desktop, and ultrawide
- **24 Unique Pages** — Each page designed as a standalone premium experience
- **Loading States** — Skeleton loaders and smooth transitions
- **Toast Notifications** — Non-intrusive feedback for every action

### Technical Quality
- **Zero TypeScript errors** — Strict typing throughout
- **Clean Architecture** — Zustand stores, modular components, separation of concerns
- **Performance** — Static generation, optimized images, minimal bundle
- **Accessibility** — Focus management, semantic HTML, keyboard navigation

## Getting Started

```bash
# Clone the repository
git clone https://github.com/m-sharique-sabir/luxe-store.git

# Navigate to project
cd luxe-store

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
